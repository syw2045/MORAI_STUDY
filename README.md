# MORAI Simulation STUDY

This repository is for MORAI Simulation Study
1. Camera & GPS
2. Slam & Navigation

## Environment
```
Ubuntu : 20.04.6 LTS
ROS : Noetic
MORAI : 22.R4.2
ROBOT : WeGo-ST MINI
MAP : WeGo_LAB & K-City
```

## Running
```
git clone https://github.com/syw2045/MORAI_STUDY.git
cd MORAI_STUDY && cd <workspace>
catkin_make
```
